# note自動投稿をPCスリープ中でも動かす設定

これは「GitHub Actions」という、ネット上で決まった時間に処理を動かす仕組みを使います。
あなたのPCが閉じていても、GitHub側で毎朝8:00に動きます。

## 必要なもの

- GitHubアカウント
- OpenAI APIキー
- noteにログイン済みの状態を保存した秘密情報

## できること

毎朝8:00に以下を実行します。

1. AIの最新情報を調べる
2. note記事を書く
3. noteに投稿する

## 失敗したとき

noteのログインが切れた、本人確認が出た、画面が変わった場合は止まります。
その場合は、今あるPC起動方式をそのまま使います。

## 次に必要な作業

このフォルダをGitHubのリポジトリに入れます。
そのあと、GitHubの設定画面で次の2つを「Secrets」に保存します。

- `OPENAI_API_KEY`
- `NOTE_STORAGE_STATE_B64`

`NOTE_STORAGE_STATE_B64` は、noteにログイン済みの状態を保存したものです。
パスワードをここに書く必要はありません。
